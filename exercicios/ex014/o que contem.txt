Digitar style na cabeca podendo ser sobre/sob o titulo
Por exemplo  BODY e usa o seletor {} tudo que estiver no corpo sera modificado, quando em H1 utiliza-se o seletor {} tudo que estiver com o h1 sera modificado e assim por diante.

<title>Estilos Locais/Internos</title>
    <style>
        body {
            background-color: aquamarine;
            font-family: Arial, Helvetica, sans-serif;
            font-size: 20px;
        }
        h1 {
            color: blue;
            background-color: lightblue;
        }
        h2 {
            color: darkred;
        }
        p {
            text-align: justify;
        }
    </style>


Essa tecnica e excelente para modificar documentos com poucas paginas, ser for muitas nao e tao eficiente.
